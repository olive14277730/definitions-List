What these Terms Mean

Accessibility
Is the ability of web design to be usable across all devices and by people with disabilities.
Anchor text
Clickable text in a hyperlink.
Back-Link
Incoming hyperlink from one web page to another.
Browser Cache
A repository for stored data, that is used to quicken the rate at which to retrieve data.
Deprecated
This is a status applied to software features to indicate that it is outdated.
GUI
Graphic User Interface, is an interface which allows users to interact with electronic devices.
Non-Breaking Space
A space character that prevents an automatic line break to a new line.
RSS
Rich Site Summary, is a format of delivering regularly changing web content.
